# 鋒兒AI資訊系統

智能管理您的影片和圖片收藏，支援智能分類和快速搜尋的多功能資訊管理系統。

## 系統功能

### 🖼️ 圖片庫管理
- 支援多種圖片格式 (JPG, PNG, GIF, WebP)
- 智能分類和標籤系統
- 快速搜尋和篩選功能
- 圖片資訊自動識別 (尺寸、檔案大小等)

### 🎬 影片庫管理
- 支援多種影片格式 (MP4, AVI, MOV, WMV)
- 影片預覽和播放功能
- 分類管理和描述系統
- 檔案大小和格式資訊顯示

### 🍎 食品管理系統
- 食品庫存管理
- 到期日期追蹤和提醒
- 購買日期和價格記錄
- 存放位置管理
- 自動狀態更新 (正常/即將到期/已過期)

### 📅 訂閱管理系統
- 各種訂閱服務管理
- 自動計算下次帳單日期
- 到期提醒功能
- 價格和週期追蹤
- 訂閱狀態管理 (啟用/暫停/取消)

### 📊 系統儀表板
- 即時統計資料顯示
- 到期提醒彙整
- 系統狀態監控
- 快速導航功能

## 技術架構

### 前端技術
- **UI 框架**: Tailwind CSS
- **圖示**: Font Awesome 6.0
- **響應式設計**: 支援各種裝置尺寸
- **玻璃擬態設計**: 現代化視覺效果

### 後端技術
- **程式語言**: PHP 7.4+
- **資料庫**: MySQL 5.7+ / MariaDB 10.2+
- **架構模式**: MVC 架構
- **安全性**: PDO 預處理語句、密碼雜湊、Session 管理

### 系統需求
- PHP 7.4 或更高版本
- MySQL 5.7 或 MariaDB 10.2 或更高版本
- Apache 2.4+ (建議) 或 Nginx
- 至少 512MB RAM
- 至少 1GB 可用磁碟空間

## 安裝說明

### 1. 環境準備
確保您的伺服器已安裝：
- PHP (啟用 PDO MySQL 擴展)
- MySQL 或 MariaDB
- Apache 或 Nginx

### 2. 檔案部署
將所有檔案上傳到您的網站根目錄 (通常是 `public_html` 或 `www`)

### 3. 資料庫設定
1. 建立 MySQL 資料庫 (資料庫名稱: `hsin_php`)
2. 設定資料庫使用者權限

### 4. 設定檔案
資料庫連線設定已在 `config/database.php` 中配置：

**本地測試環境:**
- 主機: localhost
- 帳號: root
- 密碼: (空白)
- 資料庫: hsin_php

**遠端上線環境:**
- 主機: localhost  
- 帳號: hsin_php
- 密碼: ym0Tagood129
- 資料庫: hsin_php

### 5. 執行安裝
1. 在瀏覽器中訪問 `http://您的網域/install.php`
2. 系統會自動建立所需的資料表和預設資料
3. 安裝完成後，刪除 `install.php` 檔案 (安全考量)

### 6. 登入系統
使用預設管理員帳號登入：
- 帳號: `admin`
- 密碼: `admin123`

**重要**: 登入後請立即修改預設密碼！

## 使用說明

### 圖片管理
1. 點擊「新增圖片」上傳圖片檔案
2. 設定分類和描述資訊
3. 使用搜尋功能快速找到所需圖片
4. 支援按分類篩選

### 影片管理
1. 上傳支援的影片格式
2. 系統自動識別檔案資訊
3. 可直接在瀏覽器中預覽播放
4. 分類管理便於整理

### 食品管理
1. 新增食品時設定到期日期
2. 系統自動計算剩餘天數
3. 到期提醒功能避免食物浪費
4. 支援存放位置和價格記錄

### 訂閱管理
1. 記錄各種訂閱服務資訊
2. 自動計算下次帳單日期
3. 提前提醒避免忘記續費
4. 支援不同計費週期
