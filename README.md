# 鋒兒AI資訊系統 (phpkiro20260101)

智能管理您的影片和圖片收藏，支援智能分類和快速搜尋的多功能資訊管理系統。

## 🎯 專案概述

這是一個基於 PHP 和 MySQL 的現代化資訊管理系統，提供圖片庫、影片庫、食品管理和訂閱管理等功能。系統採用響應式設計，支援無需登入直接使用，適合個人或小團隊使用。

## 系統功能

### 🖼️ 圖片庫管理
- 支援多種圖片格式 (JPG, PNG, GIF, WebP)
- 智能分類和標籤系統
- 快速搜尋和篩選功能
- 圖片資訊自動識別 (尺寸、檔案大小等)

### 🎬 影片庫管理
- 支援多種影片格式 (MP4, AVI, MOV, WMV)
- 影片預覽和播放功能
- 分類管理和描述系統
- 檔案大小和格式資訊顯示

### 🍎 食品管理系統
- 食品庫存管理
- 到期日期追蹤和提醒
- 購買日期和價格記錄
- 存放位置管理
- 自動狀態更新 (正常/即將到期/已過期)

### 📅 訂閱管理系統
- 各種訂閱服務管理
- 自動計算下次帳單日期
- 到期提醒功能
- 價格和週期追蹤
- 訂閱狀態管理 (啟用/暫停/取消)

### 📊 系統儀表板
- 即時統計資料顯示
- 到期提醒彙整
- 系統狀態監控
- 快速導航功能

## 技術架構

### 前端技術
- **UI 框架**: Tailwind CSS
- **圖示**: Font Awesome 6.0
- **響應式設計**: 支援各種裝置尺寸
- **玻璃擬態設計**: 現代化視覺效果

### 後端技術
- **程式語言**: PHP 8.3+
- **資料庫**: MySQL 8.4+ / MariaDB 10.2+
- **架構模式**: MVC 架構
- **安全性**: PDO 預處理語句、密碼雜湊、Session 管理

### 系統需求
- PHP 8.3 或更高版本
- MySQL 8.4 或 MariaDB 10.2 或更高版本
- Apache 2.4+ (建議) 或 Nginx
- 至少 512MB RAM
- 至少 1GB 可用磁碟空間

## 🚀 快速開始

### 1. 克隆專案
```bash
git clone https://github.com/goldshoot0720/phpkiro20260101.git
cd phpkiro20260101
```

### 2. 環境準備
確保您的伺服器已安裝：
- PHP (啟用 PDO MySQL 擴展)
- MySQL 或 MariaDB
- Apache 或 Nginx

### 3. 資料庫設定
資料庫連線設定已在 `public_html/config/database.php` 中配置：

**本地測試環境:**
- 主機: localhost
- 帳號: root
- 密碼: (空白)
- 資料庫: hsin_php

**遠端上線環境:**
- 主機: localhost  
- 帳號: hsin_php
- 密碼: ym0Tagood129
- 資料庫: hsin_php

### 4. 安裝系統
1. 訪問 `http://您的網域/public_html/server-info.php` 檢查系統狀態
2. 執行 `http://您的網域/public_html/migrate-data.php` 建立資料表
3. 可選：執行 `http://您的網域/public_html/demo-data.php` 建立示範資料
4. 開始使用：`http://您的網域/public_html/`

## 🛠️ 開發工具

專案包含多個實用的開發和維護工具：

- **server-info.php** - 系統狀態檢查
- **debug-db.php** - 資料庫連線除錯

- **migrate-data.php** - 資料遷移工具
- **demo-data.php** - 示範資料產生器

## 📊 資料遷移

系統支援從現有的 `goldshoot0720` 資料庫遷移資料：

- 自動建立新的資料表結構
- 遷移食品資料 (`food` → `foods`)
- 遷移訂閱資料 (`subscription` → `subscriptions`)
- 保持資料完整性和一致性
